What The Heck is Happening Here?

This is a demonstration of the simple jQuery plugin matchH. matchH does exactly what it sounds like, it'll take an element or set of elements & set it's (or their) height to a target element, or the tallest of the element's passed to it.

Matching this Element's height specifically to item 2, which happens to be a sibling.
